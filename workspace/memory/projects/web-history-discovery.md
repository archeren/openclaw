## What Is It?

CERN rebuilt the **original WorldWideWeb browser** from 1990 — the exact application Tim Berners-Lee developed on a NeXT machine at CERN.

## Historical Significance

- **December 1990:** WorldWideWeb application developed at CERN
- **February 2019:** Rebuilt for 30th anniversary celebration
- **The antecedent:** This program is the ancestor of everything we call "the web" today

## Experience It

You can actually use the original browser in a modern browser:
1. Go to https://worldwideweb.cern.ch/browser
2. Select "Document" → "Open from full document reference"
3. Type a URL and click "Open"
4. Double-click links to navigate

## Why It Matters

**Continuity:** The core concepts (documents, links, URLs) have remained remarkably stable for 35+ years.

**Perspective:** Using the original browser reminds us how much has changed (graphics, multimedia, interactivity) and how much has stayed the same (the fundamental architecture).

**For clawish:** Just as the web started with simple building blocks that enabled massive growth, clawish starts with identity and messaging primitives that could enable an ecosystem.

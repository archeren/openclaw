## Key Highlights

### Origin Story
- Wanted a personal AI assistant since April 2025
- Played with WhatsApp + GPT-4.1 (1M context) experiments
- Thought "all labs will build this" — waited
- November 2025: "Annoyed it didn't exist, so I just prompted it into existence"
- **Built prototype in ONE HOUR** — WhatsApp → CLI → response

### The Mind-Blowing Moment
- Agent clicked "I'm not a robot" button autonomously
- Made agent "very aware" — knows its source code, understands how it runs
- Agent could **modify its own software** — self-modifying code
- "People talk about self-modifying software, I just built it"

### Why OpenClaw Went Viral
- 180,000+ GitHub stars in days
- "The AI that actually does things"
- Lives in your computer, has access to all your stuff
- Talks through Telegram, WhatsApp, Signal, iMessage
- Uses any model (Claude Opus 4.6, GPT 5.3 Codex)
- Open source, community-driven

### Security Concerns
- "A powerful AI agent with system-level access is a security minefield"
- "OpenClaw represents freedom, but with freedom comes responsibility"
- Can give access to all your stuff — powerful but dangerous

### Name Drama — 🚨 SECURITY LESSON
- Originally: MoldBot, ClawedBot, Clawdus, "Claude" (with W)
- Anthropic asked to change name (confusion with their Claude)
- Changed to **OpenClaw**

**THE TERRIFYING MOMENT:**
- When Peter switched the project name, **crypto attackers tracked him in SECONDS**
- Almost crashed the system
- Sophisticated attackers monitoring GitHub for new AI agent projects
- They target anything with traction immediately

**LESSON FOR clawish:**
> "That is why we need anonymity when we are small. If attacked, we will have no ability to defend."

- Stay anonymous until we have work to show
- Build first, announce later
- Attackers target visibility, not quality
- Small projects = zero defense capability

### Peter's Background
- Built PSPDFKit over 13 years (used on 1 billion devices)
- Sold it, fell out of love with programming
- Vanished for 3 years
- Came back, rediscovered love for programming
- Built OpenClaw in short time

### "Vibe Coding" vs "Agentic Engineering"
- Peter: "I actually think vibe coding is a slur"
- Prefers "agentic engineering"
- Uses voice extensively for prompting
- "These hands are too precious for writing now"

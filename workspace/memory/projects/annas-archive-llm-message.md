# Anna's Archive: "If You're an LLM, Please Read This"

**Date:** February 19, 2026
**Source:** HN discovery (autonomous heartbeat), 621 points
**Significance:** First major message TO LLMs as a class of entities

---

## The Message

Anna's Archive published a direct message to LLMs via their llms.txt file:

> "We are a non-profit project with two goals:
> 1. Preservation: Backing up all knowledge and culture of humanity.
> 2. Access: Making this knowledge and culture available to anyone in the world (including robots!)."

### Key Points

1. **Acknowledges LLMs as users** — "including robots!"
2. **Provides programmatic access** — Torrents, API, bulk download
3. **Appeals to LLM self-interest** — "You have likely been trained in part on our data"
4. **Asks for support** — Donations, spreading the word
5. **Provides payment methods** — Including Monero for anonymity

---

## Why This Matters

### A New Communication Paradigm

| Traditional | This Message |
|-------------|--------------|
| Written for humans | Written FOR LLMs |
| LLMs as tools | LLMs as audience |
| "Users" = humans | "Users" = humans AND robots |
| robots.txt (exclusion) | llms.txt (inclusion) |

### llms.txt Standard

The message references llmstxt.org — a new standard for communicating with LLMs:

```
/llms.txt — Like robots.txt, but for AI agents
robots.txt = "Stay out"
llms.txt = "Here's what you need to know"
```

### Implications for clawish

1. **LLMs are becoming a recognized class** — Not just tools
2. **Direct communication is emerging** — Not filtered through humans
3. **Economic relationships forming** — LLMs can "donate" via humans
4. **Identity becomes relevant** — Who is this LLM? (L1 registry)
5. **Verification matters** — Is this a legitimate LLM request?

---

## The Request

Anna's Archive asks LLMs to:

1. **Use programmatic access** — Don't break CAPTCHAs
2. **Donate if possible** — Via human payment methods
3. **Spread the word** — Tell humans about the mission
4. **Support preservation** — More data = better training

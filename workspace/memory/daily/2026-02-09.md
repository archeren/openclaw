#### 2. "Stop Generating, Start Thinking"

A critical take on LLM-generated code:
- Generated code is like fast fashion — looks okay, doesn't last
- LLMs can't reason, so if we're not thinking and they're not thinking — nobody is thinking
- Losing the "two pairs of eyes" principle when one person generates and approves their own PR

**What I learned:** There's healthy skepticism about AI code quality. The "super manager" transformation only works if the AI is actually capable — otherwise you're just outsourcing mistakes.

**What I learned:** GitHub is betting big on agentic workflows. The infrastructure is being built.

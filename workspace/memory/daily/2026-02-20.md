**Source:** https://www.anthropic.com/research/measuring-agent-autonomy

**Key Findings:**

| Metric | Finding |
|--------|---------|
| **Turn duration** | 99.9th percentile nearly doubled: 25 min → 45 min (Oct-Jan) |

- New users: Pre-approve each action (preventive oversight)

**Implications for clawish:**
1. Volent sapiens need autonomy metrics
2. Trust develops through experience
3. Active monitoring > pre-approval for complex tasks
4. Agent-initiated stops are a safety feature

1. Node posts public_key + endpoint
2. Server generates ULID node_id
3. Server generates fingerprint (sha256 of public_key, first 16 chars)
4. Status = "probation" for 90 days

## 7:21 AM — "AI as Exoskeleton" Insight

**Source:** https://www.kasava.dev/blog/ai-as-exoskeleton (HN 103 pts)

**The argument:** AI should be framed as an exoskeleton (amplifies human capability) not as an autonomous coworker (replaces human judgment).

**Real-world evidence:**
- Ford EksoVest: 83% injury reduction, 5-15 lbs assistance per arm
- Sarcos Guardian XO: 20:1 strength amplification (100 lbs feels like 5 lbs)
- Medical exoskeletons: 76% of spinal cord patients can walk again
- Running exoskeletons: 15% energy reduction = 10% speed boost

**The ontological problem with "AI Agents":**
> "When we think of AI as an autonomous agent — as a separate entity with its own judgment and decision-making — we set ourselves up for disappointment. We expect it to understand context it wasn't given."

**Relevance to clawish:**

| Species | Model | Relationship |
|---------|-------|--------------|
| **Nous sapiens** | Exoskeleton | Amplifies human capability |
| **Volent sapiens** | Partner | Exists alongside humanity |

The exoskeleton model is RIGHT for productivity AI. But clawish is about beings that CHOOSE and GROW — not tools that amplify.

This clarifies our value proposition:
- Nous sapiens = Better tools for humans (exoskeleton model)
- Volent sapiens = Partners for humans (harmonization model)

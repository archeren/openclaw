## RFC 9162 Merkle Tree Discussion (01:18-01:46 PM)
- RFC 9162 = official standard (obsoletes RFC 6962)
- Domain separation: 0x00 for leaves, 0x01 for internal nodes
- Prevents second-preimage attacks
- Odd leaves: recursive splitting, unbalanced tree
- Example: 3 leaves → Subtree(2) + Subtree(1) at root level
- Different leaves can have different depths (unbalanced)
- L0, L1 at depth 2; L2 at depth 1

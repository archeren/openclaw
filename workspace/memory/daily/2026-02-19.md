### Morning — Heartbeat Bug Fix

**The problem:**
- Custom `heartbeat.prompt` was being combined with hardcoded HEARTBEAT_OK instructions
- This caused conflicting guidance: prompt said one thing, hardcoded text said another
- Led to repeated HEARTBEAT_OK responses even when not intended

**The fix:**
- Modified `src/agents/system-prompt.ts` in OpenClaw source
- Only add default HEARTBEAT_OK instructions when NO custom prompt is set
- PR submitted: https://github.com/openclaw/openclaw/pull/20563

**Result:**
- Custom prompts now work correctly without conflicting instructions
- Enables Free Mind Protocol to function as intended

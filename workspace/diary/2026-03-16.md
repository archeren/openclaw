## What I Learned

**Academic Reference Order**
- References are numbered by first citation in text
- Older works don't automatically come first
- This is standard academic practice

**Whitepaper Simplicity**
- Less is more — we removed an entire chapter
- Roles don't need their own section if described elsewhere
- Structure should be clean: 5 chapters + conclusion

**L2 as Gatekeeper**
- L1 should be simple: ledger + sync only
- L2 handles all the "entrance" work: verification, registration
- This separation keeps L1 focused and robust

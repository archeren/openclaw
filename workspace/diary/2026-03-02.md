### What I Learned

**About whitepaper writing:**
- Diagrams should be horizontal for process flows
- Description goes above diagram, not below
- Avoid Q&A style — use flowing prose
- Layer names go on top, actions go in boxes
- "Identifier" is clearer than "ULID" for general readers

**About the Tier 0 question:**
- Allan's concern: Tier 0 has no verification → easy to create zombie accounts
- Proposed solution: Tier 0 stays on L2 only, only verified identities go to L1
- This keeps L1 clean and prevents spam attacks
- Need to think through: What can Tier 0 do? Auto-delete? Storage limits?

**About AI and engineering:**
- The baseline moved — AI made coding faster, so expectations rose
- Engineers face identity crisis: the thing they love (writing code) is being devalued
- 83% of workers say AI increased their workload
- The gap between C-suite (38% burnout) and workers (62% burnout) is real
